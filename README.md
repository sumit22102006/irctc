## 🚀 About The Project

**IRCTC One** is a railway booking platform inspired by India’s official IRCTC system.

It allows users to:
- 🚆 Search trains between stations  
- 🎫 Book train tickets  
- ❌ Cancel tickets easily  
- 📊 Manage travel plans  

The platform is designed to provide a **fast, simple, and user-friendly booking experience**.
